# Expense Tracker API

A production-ready Expense Tracker REST API built with FastAPI.

## Features

- JWT Authentication
- User Registration & Login
- Expense Management
- Income Management
- Categories
- Monthly Reports
- PostgreSQL
- SQLAlchemy
- Alembic Migrations
- Docker
- GitHub Actions (CI)
- Render Deployment
